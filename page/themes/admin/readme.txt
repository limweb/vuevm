folder นี้สำหรับระบบหลักๆใช้ในการแสดงผล ไม่แนะนำให้ปรับแต่งใช้งานใน theme นี้.

แนะนำให้สร้าง theme ขึ้นมา แล้วตั้งชื่อ folder/file ให้ตรงกัน เพื่อปรับแต่งการแสดงผล
เช่น public/themes/colored <- theme ชื่อ colored แล้วข้างในมี folder/file ที่ต้องการปรับแต่งให้ตรงกัน ก็จะแต่งการแสดงผลได้.

==eng==
This folder for core system use as 'views', do not recommend to modify this theme.

To create new theme, create new folder in /public/themes and match the name of folders and files.
eg. public/themes/colored <- this theme name colored and there are same structure of folders and files.
